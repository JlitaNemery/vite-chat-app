## Installation
1. Clone the repository  
2. Run `npm install`  
3. Create a `.env` file and add your Firebase config  
4. Start the app: `npm run dev`  

## Testing
Run unit tests using:
```bash
npm run test
```

## Deployment
1. Build the app: `npm run build`  
2. Deploy to Firebase Hosting: `firebase deploy`
